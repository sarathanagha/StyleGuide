Buttons are used for final actions and their intent is to "complete" a user flow. For example, at the end of a form to submit it, or shown at the end of a modal dialog to close it.

DateTimePicker

A datepicker is an inline or popover calendar that allows the user to select a date or date range. It is invoked when a Text Field is selected. A date picker can also contain a time picker.

<input type="text" name="datepicker" data-bind="componentdatePicker:{myDate:myDate},value:myDate" />
For eg: var viewModel = {myDate: ko.observable(new Date("12/01/2013"))};

Menus
