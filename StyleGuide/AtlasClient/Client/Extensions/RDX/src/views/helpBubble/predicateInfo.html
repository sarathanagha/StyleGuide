Example queries:
*.* Returns all events from all event sources
*.Description LIKE hello world Returns events containing the phrase "hello world"
in property Description across all event sources
hello world Returns events containing the phrase "hello world"
*.PointValue = 3.14 Returns events with PointValue equal to 3.14
*.Status LIKE_ANY Good Bad Returns events with Status containing "Good" or "Bad"
*.PointValue > 3.14 AND
*.Status LIKE Good		 Returns events with PointValue greater than 3.14 and Status "Good"
*.PointValue > 3.14 AND
*.Status LIKE Good AND NOT
*.Description LIKE hello world		 Returns events with PointValue greater than 3.14 and Status "Good"
and Description not containing the phrase "hello world"
