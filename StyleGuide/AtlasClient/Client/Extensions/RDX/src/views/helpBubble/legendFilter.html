The filtering language is regular expressions:
hello Returns items containing "hello"
^hello Returns items starting with "hello"
hello$ Returns items ending with "hello"
h.?llo Returns items containing "h" optionally followed
by any character, then followed by "llo"
e.g. "hello", 'hkllo", "hllo"
h(el|le)lo Returns items containing "h" followed by "el" or
"le", followed by "lo"
e.g. "hello", "hlelo"
