This directory has Storage Account Configuration for Atlas MDS accounts (Prod and Test)

If you're adding a new storage account or modifying an existing one (e.g. changing keys) then you need to upload the storage account configuration to MDS service, please follow instructions at https://microsoft.sharepoint.com/teams/WAG/EngSys/Monitor/AmdWiki/MDS.EXE%20Command%20Reference.aspx#UploadStorageConfig
